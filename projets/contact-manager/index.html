Contact Manager

10 octobre 2022 • 3 jours
Projet personnel Développement web PHP JavaScript Bulma Git

Présentation du projet

Ce projet consistait en la création d'un gestionnaire de contacts. Pour cela, j'ai utilisé le langage PHP et le framework CSS Bulma.

Ce que ce projet m'a apporté

Grâce à ce petit projet, j'ai pu consolider mes connaissances en PHP. J'ai aussi pu apprendre à mettre un site web en ligne à l'aide d'OVH.

Pistes d'amélioration

  • Ajout de la fonctionnalité permettant d'ajouter plusieurs numéros de téléphone à une personne.
  • Ajout de la fonctionnalité permettant d'ajouter plusieurs adresses mail à une personne.
  • Ajout de la fonctionnalité permettant de lister toutes les entreprises et de pouvoir voir les contacts dans celle-ci.
